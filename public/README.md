# Asset Manifest

Please place your manual assets here:

## 1. Brand Assets
**Path**: `public/brand/`
- **Logo**: Place as `logo.svg` (Preferred) or `logo.png`.

## 2. Local Fonts
**Path**: `public/fonts/`
- **Format**: `.woff2` (Preferred) or `.ttf`.
- **Files**:
    - `Inter-Variable.woff2` (or similar for UI)
    - `JetBrainsMono-Variable.woff2` (or similar for Code/Data)

*The application will automatically look for these paths once configured.*
